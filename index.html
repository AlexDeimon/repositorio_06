Ejercicio 1

  1. Construya una página HTML que visualice "Hola mundo".
  2. Mejore la página centrando el texto y agregandole un color diferente.
  3. Adicione una imagen.
  4. Contruya una página HTML que tenga un icono en el navegador (favicon.ico) y un titulo en la pestaña.
  5. La página debe mostrar fecha y hora actuales.
  6. Agrege un input de tipo file para subir una imagen, esta debe cargarse a la página por medio de un botón.
  7. Coloque diferentes tipos de INPUT, incluyendo textos, números, password, fechas, select, radio, checks.
  8. Adicione un botón Crear. Adicione un botón Consultar. Adicione un botón Modificar. Adicione un botón Borrar.

Ejercicio 2

Construya un formulario web que permita registrar y validar los datos de una hoja de vida.

  1. Debe incluir un campo para la foto.
  2. Incluya secciones para datos personales, datos profesionales (hasta 5 registros), educación (3 registros) y referencias.
  3. El formulario debe tener un botón “Validar datos” que lanza un proceso de validación en el cliente. Debe validar longitudes, valores válidos, fechas correctas.
  4. Una vez todos los datos sean validos se podrá subir el formulario. Los datos se deben almacenar en una base de datos en Firebase.
  5. Agregar un componente de registro, de inicio y salida de sesión mediante Firebase Authentication.
  6. Adicional: Agregar un método de consulta de hojas de vida.
